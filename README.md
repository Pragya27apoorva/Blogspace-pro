# 🚀 BlogSpace Pro

A full-stack MERN blogging platform with:
- User authentication
- Follow system
- Personalized feed
- Mobile-first UI

## 🛠 Tech Stack
- React
- Node.js
- Express
- MongoDB

## ▶️ Run Locally

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
